# HOW TO PLAY

アイテムを駆使して迫り来る強敵と戦いダンジョンの深部を目指そう。

## 基本操作

| キー    | アクション                  |
| ------- | --------------------------- |
| a,w,s,d | 移動                        |
| i       | インベントリを開く / 閉じる |
| k       | 攻撃                        |
| u       | ダッシュ                    |
| o       | 足踏み                      |
| p       | セーブ                      |
